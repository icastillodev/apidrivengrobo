    Guía de Insumos

    Módulo para la gestión de productos consumibles vinculados por departamento.

    Guía de Gestión de Insumos Experimentales
    1. BÚSQUEDA Y FILTRADO AVANZADO

    Localice pedidos por Investigador, Departamento o ID. Utilice el selector de estados para filtrar rápidamente las solicitudes pendientes de entrega o las ya procesadas.

    2. CONTROL ADMINISTRATIVO

    Actualice el estado del pedido (Proceso, Reservado, Entregado, etc.) en tiempo real. La Aclaración Administrativa se guarda automáticamente mediante auto-save al perder el foco del campo, facilitando la agilidad operativa.

    3. MODIFICACIÓN TÉCNICA DE PRODUCTOS

    En el modal, puede cambiar el Departamento Destino y gestionar la lista de productos. Use el botón "+ AGREGAR PRODUCTO" para insertar nuevas filas desde el catálogo institucional y defina las cantidades necesarias para el pedido.

    4. SISTEMA DE NOTIFICACIONES

    El botón NOTIFICAR envía un correo formal al investigador. El sistema incluye automáticamente un desglose de los insumos solicitados y el estado actual del trámite, permitiendo una comunicación fluida sin salir de la plataforma.

    5. EXPORTACIÓN Y ARCHIVO

    Obtenga el PDF oficial de la solicitud con el detalle de productos y el historial de comunicaciones. Use la exportación a EXCEL para generar reportes consolidados por rango de fechas para auditorías y control de stock.
