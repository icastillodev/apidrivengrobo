    Guía de gestión de protocolos
    1. Filtros y búsqueda

    Use origen, ámbito (si aplica), el campo “Filtrar por” y el cuadro de texto; pulse el botón de búsqueda para actualizar la grilla.

    2. Solicitudes, alta y detalle

    “Solicitudes” abre la bandeja de trámites entrantes. “Nuevo” crea un protocolo. Clic en una fila abre la ficha para revisar o editar según permisos.

    3. Exportar a Excel

    Descarga el listado filtrado actual para informes o auditoría interna.
