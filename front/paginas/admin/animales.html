    Guía de Gestión de Animales Vivos
    1. BÚSQUEDA Y FILTROS INTELIGENTES

    Utilice la barra superior para localizar pedidos rápidamente por Investigador, Protocolo o Estado. Los filtros se aplican en tiempo real sobre la grilla.

    2. GESTIÓN DE ESTADOS Y NOTAS ADMIN

    Al abrir un pedido, puede cambiar el estado (Proceso, Reservado, Entregado, etc.). Las Aclaraciones Admin se guardan automáticamente al perder el foco del campo.

    3. CÁLCULO DE PRECIOS Y EXENCIONES

    El sistema calcula el precio total basándose en la subespecie y la cantidad. Si el pedido pertenece a una categoría exenta, el sistema mostrará automáticamente el aviso EXENTO DE PAGO (100%).

    4. NOTIFICACIONES POR CORREO

    Utilice el botón NOTIFICAR para enviar un correo automático al investigador con los detalles del pedido y sus observaciones.

    5. FICHA TÉCNICA EN PDF Y EXCEL

    El botón PDF genera una hoja oficial con el membrete institucional. El botón Excel permite descargar los datos filtrados por un rango de fechas específico.
